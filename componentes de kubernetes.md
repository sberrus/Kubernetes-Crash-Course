# Componentes de kubernetes
Kubernetes ofrece una cantidad de componentes que usamos a la hora de crear aplicaciones persistentes y escalables. 

Cada uno de estos componentes tienen unas características concretas dentro del clúster que nos ofrecen darle robustes a los contenedores que tengamos desplegados.

Los componentes principales son:

- Pod
- Service
- Ingess
- ConfigMap
- Secret
- Deployment
- StatefulSet
- DaemonSet

Cada uno de los componentes permiten configurar ciertas características para controlar cosas como los despliegies, configuraciones, secretos, comunicación entre pods, etc...
